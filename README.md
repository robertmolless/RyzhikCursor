# Рыжик и Старый Загородный Дом

Атмосферная cozy life-sim про доброго рыжего кота, который оживляет старый загородный дом.

Никаких боссов. Никакого фарма. Только тёплый летний вечер, светлячки, кассеты, далёкая гитара — и место, в которое хочется возвращаться.

Готово к запуску в браузере (GitHub Pages) и подготовлено к будущему переносу в Telegram Mini Apps.

---

## Стек

- **React 18 + TypeScript + Vite** — UI, тулинг, сборка
- **Phaser 3** — основной игровой движок (мир, анимации, физика, частицы)
- **Zustand** — store состояния игры с автосохранением в `localStorage`
- **TailwindCSS** — UI стилизация (glassmorphism, мягкие панели)
- **Howler.js + Web Audio API** — процедурно синтезируемый ambient + кассеты
- Процедурная графика, сгенерированная во время выполнения (никаких внешних ассетов)

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # сборка в dist/
```

- ✦ **10 NPC** с уникальной внешностью, диалогами, рассписанием перемещения и системой дружбы.
- ✦ **5 квестов** из основного дизайн-документа, проходимых end-to-end: «Старая кассета», «Фото со светлячками», «Пропавший медиатор», «Потерянные наклейки», «Колокольчик луны».
- ✦ **Полный цикл времени суток** с плавной интерполяцией цветов (утро → закат → ночь).
- ✦ **Погода**: ясно, облачно, дождь, гроза с молниями, туман — меняется каждую игровую минуту.
- ✦ **Светлячки** ночью с механикой следования за котом и подведения их к пруду.
- ✦ **6 кассет** — каждая разблокирует уникальное процедурное настроение музыки (lo-fi, акустика, рок, эмбиент, пианино).
- ✦ **Процедурный аудиодвижок** на Web Audio API: пэд, сверчки, дождь, генеративная мелодия по «настроению» текущей кассеты.
- ✦ **Финальная сцена "Ночь Светлячков"** автоматически запускается, когда собрано большинство кассет.
- ✦ **Фоторежим** с фильтрами, виньеткой, плёночным зерном.
- ✦ **Автосохранение** в `localStorage` каждые 30 секунд.

## Архитектура

```
src/
  App.tsx                # верхний оркестратор (UI слои + Phaser)
  main.tsx
  index.css              # tailwind + cozy components
  game/
    PhaserGame.tsx       # React-обёртка Phaser
    config.ts            # конфиг Phaser, размеры мира
    scenes/
      BootScene.ts
      PreloadScene.ts    # рисует ВСЮ графику процедурно и грузит в Phaser
      YardScene.ts       # основная игровая сцена (двор)
    objects/
      Cat.ts             # Рыжик: ввод + AI поведение
      NPCObject.ts       # NPC: wander AI, dialog hook
    systems/
      LightingSystem.ts  # time-of-day overlay + god rays + vignette
      WeatherSystem.ts   # дождь, гроза, туман, молнии
      FireflySystem.ts   # ночные светлячки + механика следования
      AudioEngine.ts     # генеративный звук
    art/
      paint.ts           # утилиты canvas
      catSprites.ts      # атлас спрайтов кота
      npcSprites.ts      # атлас NPC
      worldArt.ts        # дом, дерево, костёр, кассета, ...
  ui/                    # React UI слои (HUD, Dialog, Журнал, Кассеты, Фото, Финал)
  store/useGameStore.ts  # zustand store: время, квесты, дружба, кассеты, dialog, save
  data/                  # NPCs, квесты, кассеты, диалоги
  types/index.ts
```

## Деплой на GitHub Pages

Репозиторий называется `RyzhikCursor` → базовый путь `/RyzhikCursor/`.

В репо есть `.github/workflows/deploy.yml` — он:
1. Запускается на push в `main`.
2. Делает `npm ci && npm run build` (с `VITE_BASE=/RyzhikCursor/`).
3. Публикует `dist/` на GitHub Pages.

Для активации в репо нужно:
1. Settings → Pages → Source: **GitHub Actions**.
2. Push в `main` → игра окажется на `https://<owner>.github.io/RyzhikCursor/`.

## Будущий порт в Telegram Mini Apps

Подготовлено заранее:

- В `index.html` уже подключён `telegram-web-app.js` и в `App.tsx` вызываются `tg.ready()` + `tg.expand()`.
- Для сборки под Telegram запустите:
  ```bash
  VITE_BASE=/ npm run build
  ```
  и захостьте `dist/` на любом HTTPS. Бот должен указывать на этот URL как Web App.
- В Zustand store специально нет server-side зависимостей — всё работает локально через `localStorage`.
- Архитектура подразумевает простое добавление:
  - real-time clock (просто сменить `timeScale = 1` и привязать к `Date`),
  - ежедневных событий (через `flags`),
  - визитов на дворы друзей (новая сцена + удалённый ID),
  - idle activities (Cat AI уже самостоятелен).

## Расширение

Архитектура полностью модульна. Дальнейшие добавления делаются точечно:

- **Новая локация (Лес, Пруд, Теплица, Дом изнутри)** → добавить scene в `src/game/scenes/`, зарегистрировать в `config.ts`, добавить переход в `YardScene` (взаимодействие с дверью/тропой).
- **Новый NPC** → добавить запись в `src/data/npcs.ts` + диалоги в `src/data/dialogs.ts`.
- **Новый квест** → запись в `src/data/quests.ts` + обработчик шагов в YardScene (или вынести в `QuestSystem`).
- **Новая кассета и мелодия** → запись в `src/data/cassettes.ts`, можно расширить moods в `AudioEngine.startMusic`.
- **Реальные звуковые файлы** → загружать через Howler в `PreloadScene` и передавать в `AudioEngine`, который продолжит работать с тем же API.
